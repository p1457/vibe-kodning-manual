# Ordlista

Förklaringar av termer som används i manualen.

---

## A

**API (Application Programming Interface)**
: Ett sätt för program att kommunicera med varandra. Claude Code använder Anthropics API för att prata med Claude.

**API-nyckel**
: En hemlig kod som identifierar dig när du använder en tjänst. Ska hållas hemlig som ett lösenord.

**Automation**
: En regel som får saker att hända automatiskt. I Home Assistant triggas en automation av en händelse och utför sedan åtgärder.

---

## B

**Bash**
: Ett skal (shell) för terminal. Vanligt på macOS och Linux.

**Branch**
: En gren i Git - en parallell version av ditt projekt där du kan experimentera utan att påverka huvudversionen.

---

## C

**CLI (Command Line Interface)**
: Textbaserat gränssnitt där du skriver kommandon istället för att klicka.

**Clone**
: Att kopiera ett Git-repository till din dator.

**Commit**
: En sparad "snapshot" av ditt projekt i Git. Varje commit har ett meddelande som beskriver ändringarna.

**CSS (Cascading Style Sheets)**
: Språk för att beskriva utseende på webbsidor (färger, storlekar, layout).

---

## D

**DAX (Data Analysis Expressions)**
: Formelspråket i Power BI för att skapa beräkningar.

**Dependency**
: Ett paket/bibliotek som ditt projekt behöver för att fungera.

**Deploy**
: Att publicera/lägga upp din app så den blir tillgänglig för andra.

---

## E

**Entity**
: I Home Assistant är en entity en enskild "sak" som kan övervakas eller styras (lampa, sensor, etc.).

---

## G

**Git**
: Versionshanteringssystem som spårar ändringar i filer.

**GitHub**
: Webbplattform för att lagra och dela Git-projekt.

**GUI (Graphical User Interface)**
: Grafiskt gränssnitt med fönster, knappar och menyer.

---

## H

**HTML (HyperText Markup Language)**
: Språk för att strukturera innehåll på webbsidor.

**Hook**
: En funktion som körs vid en viss tidpunkt eller händelse.

---

## I

**IDE (Integrated Development Environment)**
: Program för att skriva kod, t.ex. Visual Studio Code.

**Integration**
: I Home Assistant är en integration kopplingen till en viss typ av enheter eller tjänster.

---

## J

**JavaScript**
: Programmeringsspråk som körs i webbläsaren och ger webbsidor interaktivitet.

**JSON (JavaScript Object Notation)**
: Dataformat som ofta används för konfiguration och API-svar.

---

## L

**Localhost**
: Din egen dator som server. `http://localhost:3000` är en lokal webbserver på port 3000.

---

## M

**M (Power Query Formula Language)**
: Språket som Power Query använder under huven.

**Markdown**
: Enkelt textformat för formatering. CLAUDE.md använder Markdown.

**Measure**
: En formel i Power BI som beräknar värden dynamiskt.

**Merge**
: Att slå ihop två branches i Git.

---

## N

**NAS (Network Attached Storage)**
: Nätverksansluten lagring - en "hårddisk i nätverket".

**Node.js**
: JavaScript-miljö som kan köras utanför webbläsaren. Claude Code är byggt med Node.js.

**npm (Node Package Manager)**
: Verktyg för att installera och hantera Node.js-paket.

---

## P

**PATH**
: En lista med mappar där operativsystemet letar efter program.

**PowerShell**
: Microsofts moderna terminal för Windows.

**Prompt**
: Din text/fråga till Claude (eller annan AI).

**Pull**
: Hämta ändringar från en remote (t.ex. GitHub) till din lokala kopia.

**Push**
: Skicka dina commits till en remote (t.ex. GitHub).

---

## R

**Remote**
: En kopia av ditt Git-repository på annan plats (vanligtvis GitHub).

**Repository (repo)**
: En mapp med Git-versionhantering - innehåller alla filer och all historik.

---

## S

**Script**
: En fil med kod som utför en uppgift.

**Shell**
: Program som tolkar dina terminalkommandon.

**SSH (Secure Shell)**
: Säkert sätt att ansluta till andra datorer över nätverk.

**Staging**
: I Git - att markera ändringar som ska ingå i nästa commit.

---

## T

**Terminal**
: Program för att skriva textkommandon till datorn.

**Token**
: En enhet som AI-modeller använder för att mäta text (ungefär 4 tecken per token på engelska).

**Trigger**
: Det som startar en automation i Home Assistant.

---

## V

**Variable (variabel)**
: En namngiven behållare för ett värde i programmering.

**VBA (Visual Basic for Applications)**
: Programmeringsspråk för makron i Excel.

**Versionhantering**
: System för att spåra ändringar i filer över tid.

---

## Y

**YAML (YAML Ain't Markup Language)**
: Textformat för konfiguration, vanligt i Home Assistant. Känsligt för indentation.
