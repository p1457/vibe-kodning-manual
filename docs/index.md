# Vibe-Kodning Manual

> Lär dig programmera genom naturlig konversation med Claude Code

---

## Välj din nivå

Manualen finns i tre nivåer. Välj den som passar dig bäst - du kan byta när som helst!

=== "Nybörjare"

    **Passar dig som:**

    - Aldrig har använt terminalen
    - Är ny till programmering och versionhantering
    - Vill ha steg-för-steg-instruktioner med förklaringar

    Nybörjarnivån förklarar allt från grunden och tar inget för givet.

=== "Standard"

    **Passar dig som:**

    - Har grundläggande datorvana
    - Kanske har sett kod förut men inte skrivit själv
    - Vill ha tydliga instruktioner utan överkurs

    Standardnivån är balanserad - praktisk utan att bli för teknisk.

=== "Djupdykning"

    **Passar dig som:**

    - Har viss erfarenhet av terminal eller programmering
    - Vill förstå varför, inte bara hur
    - Uppskattar tekniska detaljer och bakgrund

    Djupdykningsnivån ger mer kontext och avancerade tips.

## Vad är vibe-kodning?

**Vibe-kodning** är ett nytt sätt att programmera där du beskriver vad du vill åstadkomma i naturligt språk, och AI:n skriver koden åt dig. Du behöver inte kunna syntax eller memorera kommandon - du behöver kunna *kommunicera*.

```
Du: "Skapa en Home Assistant automation som tänder hallampan
     när ytterdörren öppnas efter solnedgång"

Claude Code: Här är automationen...
```

## Välj operativsystem

=== "Windows"

    Instruktionerna i denna manual visar Windows-kommandon som standard.
    Du använder troligen **PowerShell** eller **Windows Terminal**.

=== "macOS"

    Byt till denna flik för att se macOS-specifika kommandon.
    Du använder **Terminal** eller ett alternativ som iTerm2.

=== "Linux"

    Linux-kommandon visas här. De flesta liknar macOS.
    Du använder din distributions standardterminal.
