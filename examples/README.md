# Exempelprojekt

Denna mapp innehåller kompletta exempelprojekt som du kan använda för att öva med Claude Code.

## Innehåll

### 1. `webbapp-todo/`
En enkel att-göra-app byggd med HTML, CSS och JavaScript.

**Prova:**
```bash
cd examples/webbapp-todo
claude
```

**Uppgifter att testa:**
- "Lägg till möjlighet att redigera uppgifter"
- "Gör appen responsiv för mobil"
- "Lägg till kategori-filter"

---

### 2. `home-assistant/`
Exempel på Home Assistant-automationer och konfigurationer.

**Prova:**
```bash
cd examples/home-assistant
claude
```

**Uppgifter att testa:**
- "Skapa en automation som dimmar lamporna kl 22:00"
- "Lägg till en notifikation när tvättmaskinen är klar"
- "Optimera automationerna för bättre läsbarhet"

---

### 3. `power-query/`
Power Query M-kod exempel för datatransformationer.

**Prova:**
- Kopiera koden till Power Query Editor
- Be Claude förklara eller modifiera

**Uppgifter att testa:**
- "Förklara vad denna transformation gör"
- "Lägg till felhantering"
- "Optimera för bättre prestanda"

---

### 4. `dax-measures/`
DAX-measures för Power BI.

**Prova:**
- Kopiera measures till Power BI
- Be Claude förklara eller skapa nya

**Uppgifter att testa:**
- "Förklara denna measure steg för steg"
- "Skapa en measure för rullande 12-månaders snitt"
- "Optimera för bättre prestanda"

---

## Hur du använder exemplen

1. **Öppna terminalen** i exempelmappen
2. **Starta Claude Code:** `claude`
3. **Utforska:** "Förklara vad detta projekt gör"
4. **Experimentera:** Prova uppgifterna ovan eller hitta på egna!

## Tips

- Börja med att be Claude förklara koden
- Gör små ändringar först
- Använd Git för att spåra dina experiment: `git init`
